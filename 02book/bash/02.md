## 1 コマンドと組み合わせについて
 
### 1-1 pipe( パイプ, | )コマンド
pipe（パイプ）コマンドに代表されるように、シェルでのコマンドは繋げて実行する事ができます。

```
表記は |
| をパイプコマンドといいます。
```

下記の動きを見てもらえば、seqコマンドで1〜5までの出力に対して,pipeコマンドでgrep検索をかけて、1を
出力してるのがわかります。

```
$ seq 1 5
1
2
3
4
5

$ seq 1 5 | grep 1
1

```
